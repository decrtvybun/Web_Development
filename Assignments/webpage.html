Python Programming

Over the past few weeks I have learnt about Python which is a language used to write simple programs and also create complex programming solutions.

In programming, data type is an important concept. Python has the following data types built-in by default, in these categories::

DATA TYPE ENTAILS
Strings This are either words or alphanumerics enclosed in quotations (either double or single) eg:
x='Hello World'
Intergers This are mostly numbers and this are not enclosed eg:
x=20
Floats This are numbers with decimals and just like numbers, they are not enclosed
x=20.5
Booleans This are data types which can either be true or false, eg:
            a = 200
            b = 33
            
            if b > a:
              print("b is greater than a")
            else:
              print("b is not greater than a")
